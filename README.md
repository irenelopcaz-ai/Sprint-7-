El objetivo de este proyectofue proporcionar posibilidades de practicar las tareas habituales de la ingeniería de software, para aumentar y complementar las habilidades en el campo de los datos y así poder convertirme en un candidato más atractivo para posibles empleadores.

Esta aplicación web crea un histograma y de un gráfico de dispersión para el conjunto de datos de anuncios de venta de coches. 

La liga de la aplicación se encuentra en la siguiente liga: https://sprint-7-plmw.onrender.com
